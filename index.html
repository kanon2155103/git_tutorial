CONFLICT

pull request

git status: 状態の確認

git diff: 変更内容の確認

git log: 変更履歴の確認

git rm: ファイルの削除

git mv: ファイル名の変更・移動

git checkout --: ファイル変更の取り消し

git reset: addの取り消し

git commit --amend: 直前のコミットの取り消し

git remote: リモートリポジトリの情報確認

git remote add [remote name] [remote url]: リモートの追加 -v: URL表示

git fetch [remote name] [branch name] + git merge [remote name/branch name]: リモートリポジトリから情報を取得 + ローカルに反映

git pull [remote name] [branch name]: 上記を一度にできる

git remote show [remote name]: リモートの詳細情報の表示

git remote rename [old temote name] [new remote name] : リモート名の変更

git remoter rm [remote name]: リモートの削除

git branch: 今いるブランチの確認 -a: 全てのブランチを表示

git branch [branch name]: ブランチの追加

git checkout [existing branch name]: ブランチの移動 -b [new branch name]: ブランチの新規作成・移動

git merge [branch name to import]: 他のブランチ内容を取り込める

git branch -m [branch name]: ブランチの変更

git branch -d [branch name]: ブランチの削除 -D: 強制削除
